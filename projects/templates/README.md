# Project Starter Templates

These templates provide a starting point for common project types.

## How to Use

1. **Choose a template** that matches your project idea
2. **Copy the template** to your own Colab notebook
3. **Follow the steps** and fill in the TODOs
4. **Adapt the code** to your specific data and requirements
5. **Don't be afraid to modify** - templates are starting points, not requirements

## Available Templates

- **Campus Info Assistant**: RAG over campus documents with safety guardrails
- **Study Helper**: RAG + Agent system for course materials
- **Local Data Explorer**: RAG system for exploring local/civic data

## Customizing Templates

- Replace example data loading with your own
- Adjust safety guardrails for your domain
- Modify Gradio UI to match your needs
- Add your own experiments for PM3

## Connection to Project Milestones

- **PM1**: Use templates to get started quickly
- **PM2**: Add RAG + Agent integration
